RoofClassify - Documentation

Description : A QGIS Tool for Automatically Identifying Asbestos Roofing A QGIS tool that is conceived for automatically identifying buildings with asbestos roofing. Its task is to identify asbestos cladding in remotely sensed images, but not to classify the different types of roofing. For more detailed information on how it works: https://www.mdpi.com/2220-9964/8/3/131.
Author and contributors: Alessandro Bacciottini, Arjan Feta, Quy Thy Truong (Oslandia), Julien Moura (Oslandia)
Plugin version: 1.0.0
QGIS minimum version: 3.16
QGIS maximum version: 3.33
Source code: https://github.com/DINFO-UniFI/RoofClassify/
Last documentation update: 01 January 2025

Quickstart

Open the plugins GUI by clicking to its icon or from the plugin menu entry Plugins-> ClassifyRoofs.

Fill the entries with the requested paths and click ok
